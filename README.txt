Observações:

Todas as telas consistem em apenas o corpo da página, sem a repetição do cabeçalho, rodapé e demais componentes da página, uma vez que quando houver a mudança dinâmica através de código JS do conteúdo da página este será o conteúdo roteado no corpo da página principal.

Arquivos:
agendamento.html - Tela onde o cliente agenda serviços do petshop
cadastrar-adm.html - Tela para cadastrar administradores no sistema (somente administradores têm acesso)
cadastro-animais.html - Tela para cadastrar animais (administradores e usuário têm acesso)
cadastro-cliente.html - Tela para cadastro de clientes (qualquer visitante tem acesso)
cadastro-produto.html - Tela para cadastrar novos produtos no sistema (somente administradores têm acesso)
cadastrar-servico.html - Tela para cadastrar novos serviços no sistema (somente administradores têm acesso)
carrinho.html - Tela pra consultar quais produtos o usuário tem a intenção de comprar (somente usuários têm acesso)
compra-produto.html - Tela para consultar produtos disponíveis para compra (somente usuários têm acesso)
gerenciar-agendamento.html - Tela onde o administrador adiciona slots de serviços ao calendário onde o cliente faz agendamentos.
historicos-vendas.html - Tela para consultar ganhos com serviços e produtos (somente administradores têm acesso)
homescreen-admin.html - Tela para a qual o administrador é redirecionado quando seu login é bem sucedido  (somente administradores têm acesso)
homescreen-usuario.html - Tela para a qual o usuário é redirecionado quando seu login é bem sucedido  (somente usuários têm acesso)
index.html - Página inicial do sistema
login.html - Página de autenticação do sistema
remover-administrador.html - Tela para remover os administradores já existentes no sistema
remover-agendamento.html - Tela onde o administrador cancela algum serviço marcado.
venda-servico.html - Tela onde o cliente confirma a compra de um serviço.
